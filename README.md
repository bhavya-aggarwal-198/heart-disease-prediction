# ❤️ Heart Disease Risk Prediction

A Machine Learning-powered web application built using Streamlit that predicts the likelihood of heart disease based on patient health metrics.

## Overview

This project uses a trained Logistic Regression model to analyze various cardiovascular health indicators and estimate the risk of heart disease.

The application provides an easy-to-use interface where users can enter their health information and receive a risk prediction instantly.

## Features

* Interactive Streamlit web interface
* Heart disease risk prediction using Machine Learning
* Probability score for prediction confidence
* Real-time prediction results
* User-friendly health data input forms
* Educational medical disclaimer
* Clean and responsive design

## Dataset

The model was trained using a publicly available heart disease dataset obtained from Kaggle.

Dataset features include:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* Oldpeak (ST Depression)
* ST Slope

## Machine Learning Model

### Algorithm Used

* Logistic Regression

### Preprocessing

* Data Cleaning
* One-Hot Encoding
* Feature Scaling using StandardScaler

### Saved Components

* `lr_heart.pkl` → Trained Logistic Regression Model
* `scaler.pkl` → Feature Scaler
* `columns.pkl` → Training Feature Columns

## Installation

### Clone Repository

```bash
git clone https://github.com/bhavya-aggarwal-198/heart-disease-prediction.git
```

### Move Into Project Directory

```bash
cd heart-disease-prediction
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Virtual Environment

Windows:

```bash
.venv\Scripts\activate
```

Mac/Linux:

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```

or

```bash
python -m streamlit run app.py
```

The application will open in your browser at:

```text
http://localhost:8501
```
